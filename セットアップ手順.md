# 横浜市保育園マップ セットアップ手順

## ディレクトリ構成

```
hoikuen-map/
├── index.html              # メインアプリ（ブラウザで開く）
├── style.css               # デザイン
├── app.js                  # アプリロジック
├── data/                   # 生成データ置き場
│   ├── nurseries_geo.json  # geocode.py が生成
│   ├── r8_202602.json      # prepare_r8.py が生成
│   ├── r7_data.json        # prepare_past.py が生成
│   └── ...
├── raw_data/               # 横浜市からダウンロードしたCSV・Excelを置く
└── scripts/                # データ変換スクリプト
    ├── prepare_r8.py
    ├── prepare_past.py
    └── geocode.py
```

---

## Step 1: データファイルのダウンロード

### 令和8年2月 入所状況データ（CSV）
横浜市オープンデータから以下をダウンロードして `raw_data/` に配置:
- `0923_20260202.csv` — 入所児童数
- `0926_20260202.csv` — 受入可能数
- `0929_20260202.csv` — 入所待ち人数

> 横浜市オープンデータポータル: https://www.city.yokohama.lg.jp/city-info/opendata/

### 令和7年データ（Excel）
- `0932_*.xlsx` — 入所児童数
- `0933_*.xlsx` — 受入可能数
- `0934_*.xlsx` — 入所待ち人数

### 施設マスター
- `ninka-opendata.csv` — 認可保育所マスター（施設番号・住所・電話番号など）
- `syoukibo-opendata.csv` — 小規模保育マスター

pip install openpyxl

## Step 3: データ変換スクリプトの実行

```bash
cd hoikuen-map/scripts/

# 令和8年データ変換
python3 prepare_r8.py

# 令和7年データ変換
python3 prepare_past.py

# 施設マスター + 緯度経度変換（時間がかかります）
python3 geocode.py
```

> `geocode.py` は施設数 × 0.5秒 かかります（1000施設なら約8分）。
> 中断しても再実行すれば続きから再開できます。

---

## Step 4: アプリを開く

### ブラウザで直接開く（簡単）
`index.html` をダブルクリックするか、ブラウザにドラッグ＆ドロップ。

> ⚠️ ブラウザのセキュリティ制限で JSON が読み込めない場合は、ローカルサーバーを使ってください。

### ローカルサーバーを使う（推奨）
```bash
cd hoikuen-map/
python3 -m http.server 8080
```
→ ブラウザで `http://localhost:8080` を開く

## 機能説明

### 地図
- 施設を丸マーカーで表示（緑=空きあり、橙=満員、ピンク=待ちあり）
- クラスタリングで多数のピンを整理
- マーカークリックでポップアップ表示（年齢別入所状況）

### フィルター（ヘッダー）
- 施設種別: 認可保育所 / 小規模保育
- 区: 18区から選択
- 空き状況: 空きあり / 待ちあり
- 年齢: 0〜5歳（複数選択可）
- 施設名・住所のフリーワード検索

### 年度切り替え（年度バー）
- 令和5〜8年をボタンまたはスライダーで切り替え

### サイドパネル
- 施設リスト: 条件に合う施設一覧（名前・区・空き数・待ち数）
- 空きランキング: 空き数が多い順Top30
- 人気ランキング: 待ち数が多い順Top30（需要が高い施設）
- リスト行をクリックすると地図がその施設に移動

## トラブルシューティング

### 「施設マスターデータが見つかりません」と表示される
→ `geocode.py` を実行して `data/nurseries_geo.json` を生成してください

### 地図にマーカーが表示されない
→ `nurseries_geo.json` の施設に緯度経度（lat/lng）が含まれているか確認

### CSVの文字化け
→ スクリプトが自動でエンコーディングを試みます。
   うまくいかない場合は `prepare_r8.py` の `encoding` を手動で指定してください

### enmikke.jp のリンクが404になる
→ 施設番号が正しくない可能性があります。施設マスターの番号を確認してください
